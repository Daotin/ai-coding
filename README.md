# 📚 AI Coding

## 关于本仓库

一直以来，我对 AI Coding 充满热情和好奇。从最初的探索到如今的深度实践，这一年来，我持续学习各种 AI 辅助编程工具和方法论，并将它们应用到实际项目开发中。

本仓库汇集了我在 AI Coding 领域的：

- **方法论**：从 0 到 1 的实践经验和思考框架
- **实战案例**：真实项目中的 AI 应用场景
- **经验分享**：踩过的坑和总结的技巧
- **提示词技巧**：如何更高效地与 AI 协作
- **工具推荐**：实用的 AI 开发工具和工作流

希望这些内容能够帮助到同样对 AI Coding 感兴趣的朋友们，让我们一起探索 AI 赋能开发的无限可能！
